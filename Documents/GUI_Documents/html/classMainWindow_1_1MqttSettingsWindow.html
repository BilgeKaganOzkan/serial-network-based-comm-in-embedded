Detailed Description

This class provides an user interface to configure mqtt connection settings visually.

And, it saves all configurations to mqttConfig.ini file when users press connection button and get configurations and shows it to users when this class object call. Also, after press the connect button, this class connect to the MQTT server using MQTT class.
