Detailed Description

IniParser class reads the "config.ini" file and parses to create Serial Port Config Data, MQTT Client Config Data and Message Config Data, which types are respectively IniParserSerialPortConfig, IniParserMqttClientConfig and IniParserMessageConfig enum.
